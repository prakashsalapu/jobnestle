# 🚀 JobNestle - Full Stack Job Portal

JobNestle is a full-stack job portal application that connects job seekers with recruiters. It allows users to create accounts, browse jobs, apply for positions, and manage applications, while recruiters can post and manage job listings.

---

## 🌐 Live Demo

- 🔗 Frontend: https://jobnestle.vercel.app  
---

## 🏗️ Architecture

This project follows a **monorepo structure**:
```
jobnestle/
│
├── frontend/ → React (Vite) app deployed on Vercel
├── backend/ → Node.js + Express API deployed on Render
│
└── README.md
```

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- JavaScript (ES6+)
- Fetch API
- CSS

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication

### Deployment
- Vercel (Frontend)
- Render (Backend)
- MongoDB Atlas (Database)

## ✨ Features
- Email & Password Login
- OTP Login

- Modern UI
